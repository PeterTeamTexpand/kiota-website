KIOTA — Team Photos
====================

Drop your photos into THIS folder with EXACTLY these filenames:

  mark-oflaherty.jpg        (Mark O'Flaherty)
  anthony-robinson.jpg      (Anthony Robinson)
  cliff-van-belkum.jpg      (Cliff van Belkum)
  kyne-slinger.jpg          (Kyne Slinger — optional, initials shown until added)

PHOTO TIPS
----------
- Format: JPG or PNG (JPG preferred for smaller file size)
- Size: at least 200×200 px, ideally 400×400 px or larger
- Crop: square (1:1) headshot works best — the CSS auto-crops to a squircle shape
- If a photo is missing the initials fallback (MO / AR / CB / KS) shows instead

STYLE
-----
Photos are displayed in greyscale by default (editorial look).
On hover, a warm colour tint fades in to match the Kiota amber brand colour.
No Photoshop needed — the CSS handles all styling automatically.
